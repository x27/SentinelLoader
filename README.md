# Sentinel Loader

Software for runtime patch the Uniden Sentinel (BCDx36HP/UBCD3600XLT) to support the OpenScanner firmware frequency range.

### Usage:

Copy **SentinelLoader.exe** to the Uniden Sentinel directory:
* BCDx36HP:  _"C:\Program Files (x86)\Uniden\BCDx36HP Sentinel"_
* UBCD3600XLT:  _"C:\Program Files (x86)\Uniden\UBCD3600XLT_Sentinel"_

Run Sentinel using the **SentinelLoader** software.

Enjoy!
